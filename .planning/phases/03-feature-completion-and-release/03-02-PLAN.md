Create `.github/workflows/ci.yml` with this exact content:

```yaml
name: CI

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: astral-sh/setup-uv@v7
        with:
          enable-cache: true
      - name: Install dependencies
        run: uv sync
      - name: Run tests
        run: uv run pytest
```

Must create `.github/workflows/` directory first (it does not exist yet).

Per CONTEXT.md locked decisions:
- Python 3.12 only (no matrix — `uv` picks correct interpreter from `requires-python` in pyproject.toml)
- `uv sync` installs dev deps including pytest
- `enable-cache: true` for uv download cache
- Triggers on push AND pull_request to main
